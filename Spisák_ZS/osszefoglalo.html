Öröklés és Singleton Metódusok Minden létrehozott osztály egy ősosztályból, az Object-ből származik, így örökli ezen metódusait. Érdekesség, hogy minden osztály objektum is önmagában így lehetnek singleton metódusai, amelyeket az osztály objektum-példányai nem érhetnek el. class HelloClass
  def self.koszontes
    puts "Üdvözöl a ruby nyelv!"
  end
end

HelloClass.koszontes() → "Üdvözöl a ruby nyelv!"
HelloClass.new().koszontes() → NoMethodError

  c = -1
elsif a == b
  c = 0
else

end

1 <=> 1 → 0
1 <=> 2 → -1

puts 'de jó' if 1 > 0 → "de jó"
Érték visszaadása feltételvizsgálatból A feltételvizsgálat értéket is tud visszaadni. kimenet = if a < b then
  -1
elsif a == b
  0
else
  1
end

kimenet = a < b ? a : b
Unless (ha nem) Létezik az if fordítottja is, az unless is. Ez értelemszerűen akkor fut le, ha a vizsgált kifejezés hamis. unless 1 < 2
  er = 1
else
  er = 2
end

kim = unless 1 < 0 then 1 else 2 end

a = 1 unless a == 1 → ekvivalens: a = 1 if a != 1
